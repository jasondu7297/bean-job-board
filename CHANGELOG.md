# Changelog

## Version 2 — comprehensive live board

- Removed the ten-job practical ceiling; those roles are now only an immediate offline fallback.
- Added 16 high-volume U.S. new-grad feeds and more than 90 configured source connectors.
- Added automatic live loading inside the standalone HTML, with raw GitHub and jsDelivr fallback endpoints.
- Added source-health reporting, a manual cache-bypassing refresh, pagination, and source filtering.
- Added daily snapshot commits so scheduled builds can report jobs first seen since the prior refresh.
- Added fuzzy duplicate merging and preference for official employer records over aggregator records.
- Expanded research detection and generated role-specific cold emails.
- Added safety and quality filters for senior, internship, postdoctoral, non-U.S., stale, and restricted-sector listings.
